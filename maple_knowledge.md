### 장비란?
슬롯: 모자·상의·하의(또는 한벌옷)·신발·장갑·망토·어깨장식(견장)·얼굴장식·눈장식·귀고리·**반지 4개**·**펜던트 2개**·벨트·훈장·뱃지·포켓·엠블렘·무기·보조무기(+안드로이드, 기계심장).
반지는 연금술사의 반지를 제외하고 전부 고유장착이라 같은 반지를 중복 착용할 수 없다.
두 번째 펜던트 슬롯은 별도 확장이 필요하다.

### 아이템
같은 이름 아이템도 잠재·에디·추옵·스타포스·가횟·타월드 여부에 따라 가격이 수백 배 차이 난다.
옥션 `시세검색`은 최근 3개월 체결가를 보여준다.
세부검색은 조건 최대 5개이며, 잠재/에디 옵션은 합산되지 않으므로 원하는 줄 수만큼 조건을 따로 걸어야 한다.
옥션 판매 수수료는 5%, MVP 스페셜 또는 프리미엄PC방은 3%. 판매자가 부담한다.

### 추가옵션
추가옵션은 아이템이 생성될 때 부여되며, 등급은 일반 아이템 1~5단계, 보스 전리품 3~7단계(일반 +2)다.
유저는 최고 수치부터 1추, 2추, … 로 부른다.

| 수단 | 부여 범위 (일반/보스) | 기존/변경 선택 | 비고 |
|---|---|---|---|
| 강력한 환생의 불꽃 | 1~4 / 3~6 | X | 1추 부여 불가 |
| 영원한 환생의 불꽃 | 2~5 / 4~7 | X | |
| 검은 환생의 불꽃 | 2~5 / 4~7 | O | 최고단계 1% |
| 심연의 환생의 불꽃 | 3~5 / 5~7 | O | 최고단계 3% |
| 타오르는 환생의 불꽃 | 1~4 / 3~6 | O | |
| 추가옵션 재설정(메소) | 2~5 / 4~7 | O | 300만 메소/회 |

- 환생의 불꽃 또는 메소 재설정을 사용하면, 가위 사용 가능 횟수가 없던 아이템에 **가횟 10회가 부여되고 장착 시 교환불가가 된다.** 일반 환생의 불꽃은 5회.
- 검환불·심환불·타오르는 환불은 BEFORE를 선택해도 가횟이 부여된다.
- 가위를 사용해 1회 교환 가능 상태가 된 아이템에는 카르마 환생의 불꽃을 사용할 수 없다.
- 제네시스 무기는 2차 해방 전까지 추가옵션을 재설정할 수 없다.

### 잠재능력
레어 / 에픽 / 유니크 / 레전더리가 있다.
레전더리면 첫째줄에 레전더리 옵션, 2·3번째 줄에는 유니크 옵션이 뜬다. 유니크일때 (유/에/에).
낮은 확률로 2·3번째 줄에 1번째 줄에서만 뜨는 옵션이 뜨는데, 이것을 이탈옵션이라고 부른다.

**이탈 확률**

| 재설정 수단 | 2번째 줄 | 3번째 줄 |
|---|---|---|
| 잠재능력 재설정 / 블랙 큐브 | 1/5 | 1/20 |
| 에디셔널 재설정 (레전드리) | 1/201 | 1/201 |
| 에디셔널 재설정 (유니크/레어) | 1/51 | 1/51 |
| 에디셔널 재설정 (에픽) | 1/21 | 1/21 |

- 잠재·에디셔널 모두 메소로 재설정하며, 등급 상승 보장(천장)이 있다.
- `메멘토 큐브`는 착용 200제 이하 아이템에만 사용 가능하고 천장이 없다. 사용 시 **메멘토 각인** 상태가 되어 30일간 거래 불가, 가위·쉐어네임택·전승 사용 불가.

**기타**
- 크리티컬 데미지: 장갑 잠재 레전드리 8%(90제 이상). 에디는 장갑 3%, 그 외 부위 1%.
- 쿨타임 감소: 모자 잠재 레전드리 −1초 또는 −2초(각 50%), 에디 레전드리 −1초. 10초 이하 스킬에는 1초당 5%만 적용, 5초 이하 스킬에는 미적용. 현존 최고 수치는 8초.
- 아이템 드롭률 / 메소 획득량: 잠재 레전드리 20%, 에디 레전드리 5%. 아획은 아이템으로 최대 200%, 메획은 100%까지 적용. 드랍 43%를 확보하면 메소 주머니가 100% 드롭된다.
- 공격력 고정치(에디, 방어구·장신구): 레어 +10/+11, 에픽 +11/+12, 유니크 +14/+15, 레전드리 +16/+17.
- 9레벨당 스탯(에디): 유니크 +1, 레전드리 +2. 288레벨 기준 각각 32, 64.
- 크뎀 3줄(크크크) 확률은 0.001%. 기댓값은 200제 4.5조 메소, 250제 5조 메소.

### 무기류/방어구/장신구에 뜨는 잠재능력 차이
- **공퍼·데미지·방무**: 무기·보조·엠블렘에만
- **보공**: 무기·보조에만 (엠블렘 X)
- **메획·아획**: 귀고리·반지·얼장·눈장·펜던트에만 (벨트·견장 제외), 레전드리
- **크뎀**: 장갑 / **쿨감**: 모자(레전드리) / 오토스틸: 장갑
- **방어구·장신구 공통**: 주스탯%, 올스탯%, 공/마 고정치, 방어력, HP
- **에디 무기류**: 유효옵은 공/마%뿐. 에디 방무 최대 7%, 에디 보공 최대 20%
- **에디 방어구·장신구**: 주스탯%·올스탯% > 공/마 고정치(레어 +10) > 9레벨당 스탯

**뾰공** = 보스 데미지 20%짜리 줄. 유니크 등급 옵션이며 현재 재설정으로는 등장하지 않고 과거 아이템에만 남아 있다.
현재 무기·보조 레전드리 1줄 보공은 35% 또는 40%가 같은 확률로 나온다. 같은 "보보공"이라도 40/30 과 35/30 은 다른 물건이다.

방무는 곱연산, 보공은 합연산이다. 스탯창 방무는 92~97% 구간에서 직업별로 적정선이 갈리며, 그 이상은 보공·공퍼보다 효율이 낮다.
엠블렘에는 보공이 뜨지 않는다.
무보엠 잠재 조합 서열: 보보공 > 보공공 > 보보방 = 보방공 > 방공공.

### 스타포스
- 138제 이상 최대 30성. 강화 실패 시 하락은 없고, 15성부터 파괴가 발생한다.
- 파괴방지는 15~17성 구간에서만 적용되며 강화비용 100%를 추가 지불한다.
- 22성까지는 썬데이 메이플의 파괴확률 30% 감소가 적용된다. 23성부터는 파괴확률이 성공확률보다 높다.
- 22성까지는 주스탯과 공/마가 함께 오르고, 23성부터는 공/마만 오른다.
- 대부분의 중~고자본 유저는 22성 아이템을 거래한다.
- 제네시스 무기는 22성 고정이다. 해방 퀘스트는 17~18성 무기로 클리어 가능하다.
- 아케인셰이드 블레이드의 최대 강화 단계는 26성이다.

**장비의 흔적**
- 15성 이상에서 파괴되면 흔적이 남는다. 흔적은 잠재·에디·추옵·주문서 강화·소울·교환 조건(가횟)을 보존한다.
- 메소와 동일 장비를 지불해 파괴 직전 수치로 복구할 수 있다. 23성 이상에서 파괴된 경우 22성으로 설정된다.
- 필요 동일장비 수: ~18성 1개 / 19~20성 2개 / 21성 3개 / 22성 4개. 동일 장비 1개만 지불해 12성으로 복구할 수도 있다.

### 놀장이란?
`놀라운 장비강화 주문서`. 실패 시 흔적 없이 100% 파괴되고, 성공하면 별이 **파란색**으로 표시된다.
- 128~150제 장비에만 사용 가능. 한 번이라도 성공하면 최대 15성으로 고정되고 **스타포스 강화와 아크 이노센트 주문서를 영구히 쓸 수 없다.**
- 옵션 상승폭이 커서 놀장 12성이 스타포스 22성보다 스탯이 약간 높다.
- 1~5강은 올스탯만 오르므로 5성까지 스타포스로 올린 뒤 6강부터 놀장을 쓴다(놀타포스).
- 단종되었고 주문서 자체가 교환불가라 대리작으로만 유통된다.

### 전투력과 환산주스탯의 관계
- **전투력**(인게임 공식 지표) = 스탯 반영치 × 총 공/마 × (데미지% + 보공%) × 평균 크리티컬 보정.
  방무·레벨·포스·스킬·쿨감·직업 패시브·링크 스킬은 반영되지 않는다.
- **환산 주스탯**(유저 지표) = 방무·포스·레벨·특수옵까지 주스탯 등가로 환산한 값. 상위 보스 파티는 전투력이 아니라 환산으로 모집한다. "헥환"은 헥사코어까지 반영한 값이다.
- 수치 1당 효율: 최종데미지 > 크뎀 ≈ 공/마% > 데미지% ≈ 방무 > 주스탯% > 공/마 > 주스탯
- 크뎀 1% ≈ 주스탯 3~4%. 공/마 10 ≈ 주스탯 4%. 보공 35% ≈ 공격력 10%.
- 아획·메획·드메·쿨감은 전투력에 반영되지 않는다. 방무 잠재도 전투력을 거의 올리지 않는다.
- 환산 11만 구간에서 에테르넬·칠흑 전 부위 23성, 마이스터링·데브펜·가엔링 24~25성을 사용한다.

### 가위횟수
장착시 교환불가 아이템은 장착후에는 플래티넘 카르마의 가위를 사용해서 교환을 1번 할 수 있다. (메포 5,900)
이때 잔여 가위 사용횟수가 0이면 교환을 할 수 없기에 템 시세에 영향을 준다.
하지만 0에 가까울수록 선형적으로 가치가 감소하는 것이 아닌, 0에 가까울수록 급락적으로 감소하는 경향을 보이고
가횟0이더라도 헐값이지만 엄청 무료급으로 판매하지는 않는다.
메이플스토리의 전체 아이템은 시간이 지날수록 가치가 하락하고, 아예 교환이 안되는 아이템에도 강화를 하는 경우가 많기때문에 가횟이 낮아도 반영구적으로 사용할 아이템이라면 수요가 있다.

- 플래티넘 카르마의 가위는 월 1개까지 전액 마일리지로 구매 가능하다.
- 실버 카르마의 가위는 잡화상점에서 2,000만 메소.
- 하프 이어링·그린/레드 펜던트류는 가위 없이 교환·계정 내 이동이 가능하다. 단 환생의 불꽃을 바르면 교환불가로 바뀌면서 가횟이 생긴다.
- 혼테일의 목걸이(실버 가위), 카오스 혼테일의 목걸이·마이스터 심볼·마이스터링(플래티넘 가위)은 교환에 가위가 필요하지만 가횟은 없다.

### 교불 아이템 목록
교환 속성은 3단계다.

1. **무한 교환 가능** — 일반 드랍·제작템. 환생의 불꽃이나 추가옵션 재설정을 사용하면 2번으로 바뀐다.
2. **장착 시 교환불가** — 플래티넘 카르마의 가위로 가횟만큼 재판매 가능. 앱솔랩스, 아케인셰이드, 에테르넬, 보스 장신구, 여명/칠흑 세트, 미트라의 분노, 컴플리트 언더컨트롤.
3. **영구 교환불가(가위 사용 불가)** — 제네시스 무기, 데스티니 무기, 이벤트 반지, 특수 스킬 반지(시드링), 대부분의 훈장·뱃지, 에오스/헬리오스 귀속의 표식 아이템. 옥션에 등록되지 않는다.

### 럭키아이템이란?
이미 3개 이상 착용 중인 모든 세트에 자동으로 포함되어, 해당 부위를 대신해 세트 카운트를 채워주는 아이템.
- 2개 이상 착용해도 1개에만 적용된다. 우선순위: 4카뚝 > 스칼렛 이어링 > 스칼렛 = 제네시스 무기 > 스칼렛 링 > 스칼렛 견장.
- 해당 아이템: 카오스 벨룸의 헬름(4카뚝), 스칼렛 장비, 제네시스 무기, 데스티니 무기.
- 4카5앱, 4카5아 같은 세팅은 럭키아이템으로 성립한다.

### 세트아이템이란?
같은 세트 장비를 2개 이상 착용하면 착용 개수 단계별로 옵션이 추가되고, 서로 다른 세트의 효과는 중첩 적용된다.
- 세트 효과는 스타포스·잠재·추옵과 무관하다.
- 방어구 라인: 4카5앱 / 3카5앱 / 4카5아 / 에테르넬 3셋 기반. 에테르넬은 17~18성에서 22성 카루타와 비슷한 성능이다.
- 장신구 라인: 보스 장신구 7~9세트(9세트 = 보공 10%) → 여명의 보스 세트 → 칠흑의 보스 세트.
- 여명·칠흑 세트 효과에는 방무가 포함된다.

### 타월드 매물
타월드 매물은 이야기할 때 수수료 포함해서 얘기하지 않는다 10억+타월드 => (수수료 포함 11억) 이라고 보편적으로 하지 않음
이유는 타월드 수수료는 단순 10%의 비용증가가 아니라 메이플포인트라는 현금과 비슷한 재화가 소모되기 때문에 사용자 감각적으로 받아드리는 가치가 다름.

- 타월드 매물은 아이콘에 **W** 라벨이 붙는다. 구매 시 가격의 10% 가치에 해당하는 메이플포인트를 추가로 소모한다(최소 10메포).
- 메포 변환 비율은 매일 오전 10시 갱신되며, 09:55~10:05에는 타월드 아이템을 구매할 수 없다.
- 옥션 통합 그룹 1: 스카니아·베라·루나·제니스·크로아·유니온·엘리시움·이노시스·레드·오로라·아케인·노바 + 챌린저스 1~3
- 옥션 통합 그룹 2: 에오스·핼리오스 + 챌린저스 4
- 메소마켓 시세는 통상 1,150~1,250메포 = 1,000원이다.

### 전승 스크롤
캐시 전용, 구매 후 1회 교환 가능, 유효기간 30일.
- `추가옵션 전승 스크롤`, `잠재능력 전승 스크롤`(레전더리 한정, 99,000캐시)
- 동일한 장비끼리만 전승된다. 추출한 원본 장비는 소멸하고, 전승받은 장비는 다시 추출할 수 없다.
- 에디셔널 잠재능력은 전승되지 않는다.
- 가횟이 없는 아이템에 전승하면 가횟이 10회로 설정된다.

### 아이템 별칭
아이템 별칭은 아무 별칭이나 붙는것이 아니다.
문법: `[스타포스] [아이템/부위] [잠재] [에디] [추옵] [작] [가횟]`

- 숫자+성 → 스타포스. 파란별이면 놀장
- 숫자+% → 윗잠 % 합계 (36% 반지 = 12/12/12)
- 보/공/방 3글자 → 무보엠 윗잠 3줄 (보공/공마%/방무)
- 뾰공 → 보공 20%짜리 줄
- 올이탈 → 2·3줄 전부 이탈옵
- **에디 + 숫자열** → 에디셔널 3줄의 % 수치를 줄 순서대로 이어 붙인 것. 위 수치표로 등급을 역산한다.
  - 예) `에디 856` (200제 방어구/장신구) = 1줄 주스탯 8%(레전드리) / 2줄 올스탯 5%(유니크) / 3줄 주스탯 6%(유니크)
  - 200제 주스탯 사다리 8·6·4·2, 250제 사다리 9·7·5·3 을 기준으로 읽는다.
- 리턴 → 리턴 스크롤로 작한 것
- 급 → 실측치가 아니라 환산치

예시:
> 22성 스태프, 보보공 체인, 36% 반지, 22성 올이탈 무기, 놀장 12성, 에디 856, 뾰공, 33% 장갑, 쌍크뎀, 32%급 신발, 24성 에테르넬 공6리턴 36% 854 견장
