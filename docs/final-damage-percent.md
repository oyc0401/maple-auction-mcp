# 최종뎀 % 증가 (`finalDamagePercent`) — maplescouter 대조

경매장 매물마다 **"이 매물을 착용하면 최종 데미지(총 딜)가 몇 % 오르는가"** 를 계산해
각 매물에 `finalDamagePct` 로 채운다. maplescouter 보스 최적화의 **"최종뎀 상승량"** 과 대응한다.

환산 주스탯(`hwansanDiff`)과 달리 **스플라인(데미지→환산 곡선) 변환이 필요 없어**,
maplescouter 딜상승률과 **직접 비교·검증**할 수 있다.

## 계산 방법

```
D = StatFactor × 공격력 × (1+(데미지+보공)/100) × 크리팩터 × 방무팩터 × 최종뎀
최종뎀% = (교체 후 D / 교체 전 D − 1) × 100
```

- 데미지 공식 자체는 기존 `calc.ts` 그대로 (maplescouter 구버전식, simaple과 3중 교차확인 완료).
- 차이는 **분모(현재 스탯)에 maplescouter 표준 버프(도핑)를 반영**한다는 점 (`buffs.ts`).
  maplescouter는 넥슨 최종스탯을 그대로 쓰지 않고 표준 버프를 얹은 값을 기준으로 계산한다.
- 버프값 출처: 인게임 버프 툴팁(챌린저스·버닝·세이람·VIP·노블레스·훈련일지 등)
  + maplescouter "세부 스펙 효율(최종뎀%)" 페이지 역산.

## 정확도 (꽈숩노 · 캡틴 · 보스방어 300 기준)

스탯 1단위당 최종뎀% 를 maplescouter 효율 페이지 값과 대조:

| 항목 | 우리 | maplescouter | 오차 |
|---|---|---|---|
| %미적용 주스탯 +200 | 0.379% | 0.379% | **0.000** |
| 보총뎀 +40 | 5.319% | 5.313% | **+0.006** |
| 크뎀 +8 | 2.597% | 2.721% | **−0.124** |
| 공격력 +30 | 0.914% | 1.082% | **−0.168** |
| 방무(300) +40 | 2.061% | 1.832% | **+0.229** |

→ **전 항목 오차 0.3% 이내.**

## 한계 · TODO

- **직업/서버 일반화**: 현재 버프값은 **캡틴 / 챌린저스 서버** 기준으로 역산·검증했다
  (꽈숩노·밥이선장·마린 3인에서 주스탯 배수 ×1.154 일관 확인). 다른 직업·서버는 값 추가·테이블화 필요.
- **공격력(−0.168)**: maplescouter는 공% 적용 전 순수 공격력(F)을 기준으로 삼는데,
  현재 구현은 넥슨 최종 공격력(공% 적용 후)을 그대로 쓴다. F 기준 정식 반영 시 개선된다.
- **방무(+0.229)**: 100% 근처(near-cap)라 초민감. 챌린저스 방무 70%를 곱연산에 그대로 넣으면
  과대반영되어 제외했다(유효 방무 ~98.5% 근사). 정밀화는 maplescouter 방무 결합식 확인 필요.
- **크뎀(−0.124)**: 표준 버프 중 넥슨과 중복되는 항 트림 여지.
