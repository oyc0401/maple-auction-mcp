# 스탯 재구성 잔차 추적 (server/src/damage)

넥슨 오픈 API `final_stat`(resting)을 오라클로 삼아, 소스별 수집합과의 잔차를 기록한다.
잔차는 **조용히 흡수 금지** — 원인을 특정해서 해소하고, 모르면 유저(도메인 전문가)에게 질문.

검증 명령: `npx tsx server/src/damage/cli.ts <닉네임>` (원본은 `.nexon-raw-<닉>.json` 디스크 캐시 → 재실행 시 API 미호출)

## 검증 현황 (2026-07-11)

| 캐릭터 | 직업 | 상태 |
|---|---|---|
| 오유찬 | 카데나 | **주스탯 4종 + 공격력·마력·방무·크확·크뎀·최종뎀 잔차 0**. 데미지 -19·보공 -7 남음 |
| 티엘 | 보우마스터 | **데미지·보공·크뎀·최종뎀 잔차 0**. 주스탯·공마·크확 -10·방무 -0.22 = 챔피언의 가호(수치 미상) 추정 |

## resting(정보창) vs 전투 구분 — 확정 원칙 (2캐릭 실측)

- **resting 포함**: 장비·세트·심볼·하이퍼·어빌·유니온(공대원/전투/아티팩트/**챔피언 뱃지**)·성향 카리스마(방무)·
  무조건형 링크·직업 패시브·0차 상시(연합·여축+정축·루나·훈련일지·초월 결전·펫 세트)·메용·베이스 크확 5%.
- **resting 미포함 (전투 전용, cond 플래그)**: 조건부 링크(소콘·무아·임피 등), 조건부 스킬(위크포인트 어택/컨버징),
  **액티브 버프 스킬 전부**(샤프·쓸샤·쓸어블·레투다·영메·익스클루시브 스펠·초월 최초의 유산·크리티컬 리인포스).
- **성향 통찰력 크확은 넥슨 크확 필드에 미포함** (오유찬 실측: 빼면 정확) — 수집 안 함. 카리스마 방무는 포함.
- **공격력/마력 = floor(flat × (1+%/100))** — 오유찬 정확 일치로 공식 확정. 크확 베이스 +5, 크뎀 베이스 0.

### ❓ 미해결 — 전부 챔피언의 가호(버프, 수치 미상) 후보

오유찬(챔피언 시스템 없음)은 아래 항목이 전부 0인데 티엘(SSS×5)만 남는다 → 가호 단일 원인 가설.

| 항목 | 재구성 | API | 잔차 | 환산 (가호 추정치) |
|---|---|---|---|---|
| STR | flat 4946(올탯651 포함) | 17261 | -971 | flat -310 (=올250+부스탯60?) |
| DEX | flat 8832 | 126780 | -3320 | flat -310 (=올250+주스탯60?) |
| INT | flat 3576 | 12675 | -783 | flat -250 (올스탯) |
| LUK | flat 3353 | 11887 | -783 | flat -250 (올스탯) |
| 공격력 | floor(5517×2.92) = 16109 | 17289 | -1180 | flat -344 + 공% -3 (공%195 가정) |
| 마력 | 3554 | 3769 | -215 | flat -215 |
| 크확 | 185 | 195 | -10 | 가호 크확 10? |
| 방무 | 98.48 | 98.70 | -0.22 | 곱연산 인자 ~14.5% (가호 방무 15%?) |

→ **가호 추정 옵션**: 올스탯 250 + 주/부스탯 60 + 공마 (공344/마215는 비대칭 — 공% 3 포함 여부에 따라) + 크확 10 + 방무 15%.
  유저에게 가호 실제 공식(챔피언 등급/슬롯당 수치) 확인 요청.
