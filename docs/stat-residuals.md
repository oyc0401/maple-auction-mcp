# 스탯 재구성 잔차 추적 (server/src/damage)

넥슨 오픈 API `final_stat`(resting)을 오라클로 삼아, 소스별 수집합과의 잔차를 기록한다.
잔차는 **조용히 흡수 금지** — 원인을 특정해서 해소하고, 모르면 유저(도메인 전문가)에게 질문.

검증 명령: `npx tsx server/src/damage/cli.ts <닉네임>` (원본은 `.nexon-raw-<닉>.json` 디스크 캐시 → 재실행 시 API 미호출)

## 검증 현황

| 캐릭터 | 직업 | 상태 |
|---|---|---|
| 오유찬 | 카데나 | 주스탯 4종 잔차 0 (2026-07-11). COMMON 0차·쓸만한 배타 추가 후 재검증 필요 (API 1회) |
| 티엘 | 보우마스터 | 데미지·크뎀·최종뎀 잔차 0. 주스탯·공마·크확·보공·방무 잔차 남음 (아래) |

## 티엘 (보우마스터 Lv.296) — 잔차 현황

기준: 2026-07-11 덤프 (`.nexon-raw-티엘.json`)

### ✅ 해소됨 (원인 확정)
- **최종 데미지 0**: 결전의 의지 10 × 익스트림 아쳐리 30 × 어드퀴버 6 × 아머 피어싱 16 곱연산 = 75.83 정확.
- **데미지 0**: 조건부 링크(소콘60·무아16·시프커닝18·프프17·인텐9·임피9·전투흐름12)는 **넥슨 resting 미포함**.
  무조건형(와일드 레이지 15)만 포함 → hyper 39 + 아티팩트 15 + 와레 15 = 69 정확.
  → 링크 DB에 `cond` 플래그 구현: resting 검증 제외 / D(전투) 계산은 `includeConditional=true`.
- **크뎀 0**: 쓸샤(10/8) ↔ 본체 샤프 아이즈(20/15) **중첩 불가**(유저 확인) → 본체 보유 시 쓸만한 규칙 스킵.
  샤프 포함 시 크뎀 123.15+15 = 138.15 정확. (크뎀 베이스는 API 필드에 0으로 보임)
- **HEXA 코어 3개**(core·core_2·core_3) 전부 합산해야 함.
- **union_state_stat**(신규 유니온 전투 스탯): DEX75·공15·보공40·크뎀20·크확40·방무40 — 수집 (주스탯 flat 가정).
- **유니온 아티팩트**(/user/union-artifact): 올탯150·공마30·뎀15·보공15·방무20·크확20·크뎀4 — 수집.
- **레벨 비례 잠재** "캐릭터 기준 9레벨 당 DEX +2" → floor(레벨/9)×N (티엘 합 +320).

### ❓ 미해결 (유저 답변 대기)

| 항목 | 재구성 | API | 잔차 | 유력 가설 |
|---|---|---|---|---|
| STR | flat 4846(올탯 포함) ×3.126 +810 | 17261 | **flat -417** | 균일 -355 + α62 |
| DEX | flat 8732 ×10.713 +28870 | 126780 | **flat -407** | 균일 -355 + α52 |
| INT | flat 3476 ×3.126 +700 | 12675 | **flat -355** | 균일 |
| LUK | flat 3253 ×3.126 +610 | 11887 | **flat -355** | 균일 |
| 공격력 | flat 5487, 공% 192 | 17289 | flat ~-375, 공% -3 | 챔피언의 가호? 여축·정축 중복? |
| 마력 | flat 3524 | 3769 | flat -245 | 〃 |
| 크확 | 200 | 195 | **+5 초과** | 아래 노트 |
| 보공 | 568 | 593 | **-25** | 챔피언의 가호? |
| 방무 | 97.74 (임피 제외 후) | 98.70 | -0.96 | 곱연산 인자 누락(~42% 상당 1개 또는 조합) |

- **주스탯 노트**: noPct 합(심볼 28200 + 하이퍼 120 + 공대원 550 = 28870)은 maplescouter `mainStatAbs`와 정확 일치.
  누락은 전부 **스탯% 받는 flat** 쪽. INT·LUK 균일 -355 → 올스탯성 소스. STR +62 / DEX +52는 별도 소스.
- **챔피언의 가호** (0차, "챔피언 코인을 소모하여 30분 동안 능력치 강화" — 수치 미표기 버프):
  **resting 포함 확정** (유저 확인 2026-07-11). 남은 문제는 **수치 소스** — 스킬 텍스트에 수치가 없어
  넥슨 API 챔피언 엔드포인트(`/user/champion`? scouter도 맞추고 있으니 존재 추정)를 프로브해야 함.
  주스탯 균일 -355(+STR62/DEX52)·공마·보공 -25 잔차의 원인으로 추정.
- **크확 +5 초과**: 수집 200 = 크샷40+하스15+어빌20+데들리15+유니온50(공대원10·전투40)+아티팩트20+샤프20+훈련일지20.
  크뎀은 같은 구성으로 0이라 샤프 포함은 확실. 어떤 +5가 이중인지(혹은 훈련일지가 실제 15인지) 확인 필요.
  하이퍼패시브 "샤프 아이즈-크리티컬 레이트 +5"는 미수집 상태(넣으면 +10 초과라 제외가 맞아 보임).
- **공% -3**: scouter atkPercent 195 vs 수집 192(장비 167 + 마크맨쉽 25). 영메 4%는 버프라 미반영.
- ~~여축(30)·정축(20)~~: 둘 다 합산 **확정** (유저 확인 2026-07-11).
- ~~어둠의 다크니스 Lv.1/2/3~~ (8+10+18): **펫 스킬**, 3개 전부 중첩 **확정** (유저 확인 2026-07-11).
- **하이퍼패시브 샤프 아이즈-이그노어 가드**(방무 5% 추가): resting 포함 여부 미확정 (방무 잔차와 함께 검증).
