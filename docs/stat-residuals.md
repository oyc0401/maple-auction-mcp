# 스탯 재구성 잔차 추적 (server/src/damage)

넥슨 오픈 API `final_stat`(resting)을 오라클로 삼아, 소스별 수집합과의 잔차를 기록한다.
잔차는 **조용히 흡수 금지** — 원인을 특정해서 해소하고, 모르면 유저(도메인 전문가)에게 질문.

검증 명령: `npx tsx server/src/damage/cli.ts <닉네임>` (원본은 `.nexon-raw-<닉>.json` 디스크 캐시 → 재실행 시 API 미호출)

## resting(정보창) vs 전투 구분 — 확정 원칙 (티엘 실측)

- **resting 포함**: 장비·세트·심볼·하이퍼·어빌·유니온(공대원/전투/아티팩트/**챔피언 뱃지**)·성향·무조건형 링크·
  직업 패시브·0차 상시(연합·여축+정축·루나·훈련일지·초월·펫 세트)·메용·베이스 크확 5%.
- **resting 미포함 (전투 전용, cond 플래그)**: 조건부 링크(소콘·무아·임피 등), **액티브 버프 스킬 전부**
  (샤프 아이즈·쓸샤·쓸어블·레디 투 다이·영메·크리티컬 리인포스 등). 크뎀·크확 베이스: 크확만 +5, 크뎀은 0.

## 티엘 (보우마스터 Lv.296) — 잔차 현황

기준: 2026-07-11 덤프 (`.nexon-raw-티엘.json`)

### ✅ 잔차 0 (해소 근거)
| 항목 | 값 | 근거 |
|---|---|---|
| 데미지 | 69 | 조건부 링크 resting 제외 (hyper39+아티팩트15+와레15) |
| 보공 | 593 | **유니온 챔피언 뱃지**(/user/union-champion) 보공 25 수집 |
| 크확 | 195 | 샤프(버프) 제외 + **성향 통찰력** 100→+10 + 베이스 5 |
| 크뎀 | 138.15 | 샤프(버프) 제외 + 챔피언 뱃지 크뎀 15 (베이스 0) |
| 최종뎀 | 75.83 | 결전의지10 × 익스아처리30 × 어드퀴버6 × 아머피어싱16 |

- **유니온 챔피언 뱃지** `champion_badge_total_info`: 올스탯 100·공마 50·보공 25·크뎀 15·방무 25 — 상시 패시브.
  (챔피언의 가호=코인 소모 30분 버프는 별개, resting 포함이지만 수치가 API에 없음 — 아래 미해결)
- **성향** `/character/propensity`: 카리스마→방무 lv×0.1, 통찰력→크확 lv×0.1.
- HEXA 코어 3개(core·core_2·core_3) 합산 / union_state_stat / 아티팩트 / 9레벨당 잠재(floor(lv/9)×N) 수집.
- 쓸만한 X ↔ 본체 X 배타(스킬명 매칭). 여축+정축 합산, 펫 세트 스킬("… Lv.N" 패턴) 전부 중첩 — 유저 확정.

### ❓ 미해결

| 항목 | 재구성 | API | 잔차 | 환산 |
|---|---|---|---|---|
| STR | flat 4946(올탯651 포함) | 17261 | -971 | flat -310 (=250+60) |
| DEX | flat 8832 | 126780 | -3320 | flat -310 (=250+60) |
| INT | flat 3576 | 12675 | -783 | flat -250 |
| LUK | flat 3353 | 11887 | -783 | flat -250 |
| 공격력 | flat 5517, 공% 192 | 17289 | | flat -344~-404 (공% 195 가정 시 -344) |
| 마력 | flat 3554 | 3769 | -215 | flat -215 (마% 0 가정) |
| 방무 | 98.48 | 98.70 | -0.22 | 곱연산 인자 ~14.5% 1개 (또는 5%+10% / 6%+9% 조합) |

- **챔피언의 가호(버프) 수치 미상**: 균일 250이 "SSS 챔피언 5명 × 올스탯 50"일 가능성. STR/DEX만 +60 추가 누락은 별도 소스.
  → 유저에게 가호 공식(등급/슬롯당 수치) 확인 요청.
- **공% -3**: scouter atkPercent 195 vs 수집 192. 공/마 flat 누락은 가호 공마일 수도.
- **방무 -0.22**: 남은 인자 후보 — 샤프 이그노어가드(5%, 단 샤프가 resting 제외라 모순), 미상 10%/14.5%.
