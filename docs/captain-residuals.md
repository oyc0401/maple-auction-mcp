검증 캐릭터: **꽈숩노**(캡틴, 챌린저스, Lv281). 형이 검증한 카데나·보우마스터는 일반서버라
챌린저스 전용 소스가 엔진에 없었음.

### 1. STR/DEX 특화 잔차 (STR +65, DEX +285) — 파이렛 블레스 해적 추가스탯 추정
- 파이렛 블레스 효과창엔 "올스탯 30"만 뜨지만, 스킬 설명에 **"7레벨: 모험가 해적 직업군은 더 많은 스탯 증가"** — 해적이 받으면 API 미표시 추가 주스탯 보너스가 붙는 것으로 추정(버닝BEYOND가 스킬창 미표시였던 것과 동종).
- STR·DEX 둘 다 **더 필요**(덧셈)라서 단순 힘↔민첩 스왑 아님(스왑은 부위별로 돌려보면 DEX −967로 오히려 손해).
- **해야 할 것:** 인게임 파이렛 블레스 버프 툴팁의 실제 힘/민첩 수치 확인 → `collectLinkSkills`에서 해적 보너스 반영. (유저 확인 대기)

### 2. 공격력 +268 / 마력 +75 잔차
- 파이렛 블레스는 공격력 미부여 → 별개 소스.
- 크리/보스 계열은 전부 잔차 0이라 그 계열 버프/도핑은 없음. 공격력·마력만 남음.
- **해야 할 것:** 스탯창(넥슨 API) 스냅샷에 공격력 도핑(세이람·명예 등) 여부 확인, 또는 미수집 상시 공격력 소스 특정. (유저 확인 대기)

### 3. 검증 직업 회귀 확인
- `linkSkill.ts` 파이렛 블레스 룰의 `mult:3`(최대 3중첩 가정)이 일반서버 캐릭에서 과다계상될 수 있음 — 스택 수가 API에 없어 하드코딩. 계정별 해적 수에 따라 달라짐.
- 5차 `collectFifthBrackets`는 이미 upstream에 존재(로프커넥트/블링크/로디드다이스 등 [패시브 효과] 걷음) — 캡틴 작업 중 재확인함.
- **해야 할 것:** 오유찬(카데나)·티엘(보우마스터) 캐시로 재구성 돌려 잔차 0 유지되는지 회귀 확인.
