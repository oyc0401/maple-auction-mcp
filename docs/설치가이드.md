# 메이플 옥션 MCP 설치 가이드

AI(Claude Code, Codex 또는 Antigravity CLI)에게 **"에디공 21퍼 아케인 체인 찾아줘"** 라고 말하면, AI가 직접 메이플스토리 경매장을 검색해서 매물을 추천해 주는 도구입니다.

코딩을 몰라도, 터미널을 한 번도 안 켜봤어도 설치할 수 있도록 순서대로 안내합니다. **위에서부터 차례대로** 따라 하면 됩니다.

## 전체 순서

1. **AI 설치** — Claude Code / Codex / Antigravity CLI **셋 중 하나만** 설치 (유료 구독 필요)
2. **MCP 설치** — Node.js 설치 후, 명령어 한 줄 복사·붙여넣기
3. **크롬 확장프로그램 설치** — 클릭 한 번
4. **실행** — 경매장 로그인하고 AI에게 말 걸기

---

## 시작 전: "터미널"이 뭔가요?

아래 과정에서 명령어를 입력하는 검은 창(터미널)을 몇 번 열게 됩니다. 여는 방법만 알면 됩니다:

- **Windows**: 키보드에서 `윈도우 키`를 누르고 → `PowerShell` 이라고 검색 → 실행
- **Mac**: `command(⌘) + 스페이스바`를 누르고 → `터미널` 이라고 검색 → 실행

명령어는 직접 타이핑하지 말고, 이 문서에서 **복사해서 터미널에 붙여넣고 Enter**를 누르면 됩니다.

## 2. MCP 설치

MCP는 AI에게 "경매장 검색 능력"을 붙여주는 부품입니다. 그 부품을 돌리려면 **Node.js**라는 프로그램이 먼저 필요합니다.

### 2-1. Node.js 설치

1. [Node.js 공식 다운로드 페이지](https://nodejs.org/ko/download)에 접속
2. 초록색 **LTS** 버전 다운로드 (Windows / Mac 자동 인식)
3. 받은 파일을 실행하고, 설치 화면에서 계속 **"다음"** → **"설치"** (아무것도 바꿀 필요 없음)

설치 확인 — 터미널을 **새로 열고**(이미 열려 있던 창은 닫기) 붙여넣기:

```
node -v
```

`v20` 이상 숫자가 나오면 성공입니다.

### 2-2. MCP 등록

자기가 설치한 AI에 맞는 명령어 **한 줄**을 터미널에 붙여넣고 Enter를 누르면 끝입니다.

**Claude Code를 설치했다면:**

```
claude mcp add --scope user maple-auction -- npx -y maple-auction-mcp
```

**Codex를 설치했다면:**

```
codex mcp add maple-auction -- npx -y maple-auction-mcp
```

> 💡 **Windows에서 등록이 안 될 때**: 명령어의 `npx` 앞에 `cmd /c` 를 붙여서 다시 시도하세요.
> 예) `claude mcp add --scope user maple-auction -- cmd /c npx -y maple-auction-mcp`

**Antigravity CLI를 설치했다면:**

Antigravity CLI는 등록 명령어가 따로 없고 설정 파일(`mcp_config.json`)에 직접 추가해야 합니다. 아래 **한 줄 전체**를 복사해 터미널에 붙여넣고 Enter를 누르면 설정 파일에 자동으로 추가됩니다:

```
node -e "const fs=require('fs'),os=require('os'),path=require('path');const dir=path.join(os.homedir(),'.gemini','config');const file=path.join(dir,'mcp_config.json');fs.mkdirSync(dir,{recursive:true});let cfg={};try{cfg=JSON.parse(fs.readFileSync(file,'utf8'))}catch(e){}cfg.mcpServers=cfg.mcpServers||{};cfg.mcpServers['maple-auction']=process.platform==='win32'?{command:'cmd',args:['/c','npx','-y','maple-auction-mcp']}:{command:'npx',args:['-y','maple-auction-mcp']};fs.writeFileSync(file,JSON.stringify(cfg,null,2));console.log('등록 완료: '+file)"
```

`등록 완료`라고 나오면 성공입니다. Antigravity CLI(`agy`)를 실행해 `/mcp`를 입력하면 maple-auction이 목록에 보입니다.

> 💡 **/mcp 목록에 안 보일 때**(구버전 CLI 등): `.gemini/antigravity-cli/mcp_config.json` 파일(홈 폴더 기준)에 아래 내용을 직접 넣어보세요.
>
> ```json
> {
>   "mcpServers": {
>     "maple-auction": {
>       "command": "npx",
>       "args": ["-y", "maple-auction-mcp"]
>     }
>   }
> }
> ```
>
> (Windows는 `"command": "cmd", "args": ["/c", "npx", "-y", "maple-auction-mcp"]` 로)

### 2-3. 넥슨 오픈 API 키 — 데미지 증감률 계산 (필수)

장비 검색 결과의 슬롯별 최종 데미지 증감률 자동 계산, `user_equip`(착용 장비 조회), `refresh_character`(특정 캐릭터 강제 재갱신)를 쓰려면 넥슨 오픈 API 키가 필요합니다.
캐릭터 정보는 macOS의 `~/Library/Application Support/maple-auction-mcp`, Windows의 `%LOCALAPPDATA%\maple-auction-mcp`, Linux의 `$XDG_DATA_HOME/maple-auction-mcp` 또는 `~/.local/share/maple-auction-mcp`에 영구 캐시됩니다.

1. [openapi.nexon.com](https://openapi.nexon.com) 에서 로그인 후 API 키를 발급받습니다 (무료).
2. 등록 명령어의 `maple-auction-mcp` 뒤에 `--api-key 발급받은키` 를 붙입니다:

```
claude mcp add --scope user maple-auction -- npx -y maple-auction-mcp --api-key YOUR_NEXON_API_KEY
```

(이미 등록했다면 지우고 다시 등록하면 됩니다: `claude mcp remove maple-auction` 후 위 명령어.
`NEXON_DEVELOPER_KEY` 환경변수로 지정해도 됩니다.)

## 3. 크롬 확장프로그램 설치

경매장은 본인 메이플 계정으로 로그인해야 검색할 수 있는데, 그 로그인을 안전하게 연결해 주는 게 이 크롬 확장입니다. (로그인 정보가 외부로 전송되는 일은 없습니다 — [개인정보 처리방침](privacy-policy.md))

1. **크롬 브라우저**에서 아래 링크 접속:

   👉 [Maple Auction MCP — 크롬 웹스토어](https://chromewebstore.google.com/detail/maple-auction-mcp/mdjabnokhboicgecmpnggabjfimelfjn)

2. **"Chrome에 추가"** 버튼 클릭 → **"확장 프로그램 추가"** 클릭

끝입니다. 별도 설정은 없습니다.

## 4. 메이플 옥션 MCP 실행 방법

### ① 크롬에서 경매장 로그인

크롬에서 [메이플스토리 경매장](https://auction.maplestory.nexon.com)에 접속해 넥슨 계정으로 로그인하세요.

> ⚠️ 사용하는 동안 **이 경매장 탭을 닫지 말고 켜두세요.** 이 탭의 로그인 세션을 통해 검색이 이뤄집니다.

### ② AI 실행

터미널을 열고, 설치한 AI를 실행합니다:

```
claude
```

(Codex를 설치했다면 `codex`, Antigravity CLI를 설치했다면 `agy`)

### ③ 말 걸기

이제 채팅하듯 물어보면 됩니다. 예시:

> 경매장 연결됐는지 확인해줘

> 에디공 21퍼 아케인 체인 검색해줘

> 내가 낄 만한 무기 환산 높은 순으로 찾아줘

> 놀장 17성 앱솔 상의 시세 알려줘

캐릭터·월드는 로그인한 계정에서 자동으로 잡힙니다. 다른 월드로 검색하고 싶으면 "OO월드 기준으로 바꿔줘"라고 말하면 됩니다.

## 문제가 생겼을 때

| 증상 | 해결 방법 |
|---|---|
| "확장 프로그램이 연결되지 않았다"고 나옴 | 크롬에서 [경매장 페이지](https://auction.maplestory.nexon.com)를 열거나 **새로고침**하세요 |
| 잘 되다가 갑자기 인증 오류가 남 | 경매장 탭을 여러 개 열었거나 다른 기기에서 접속하면 세션이 끊깁니다. 탭을 하나만 남기고 새로고침하세요 |
| `claude` / `codex` / `agy` / `node` 명령어를 못 찾는다고 나옴 | 터미널을 **닫고 새로 연 뒤** 다시 시도하세요. 그래도 안 되면 해당 프로그램 설치부터 다시 확인 |
| 검색 횟수가 소진됐다고 나옴 | 경매장 검색은 계정당 **하루 100회** 제한이 있습니다(넥슨 정책). 다음 날 초기화됩니다 |
